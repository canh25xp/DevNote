# Dev Note

> A collection of notes about basically everything throughout my journey as a S/W developer
> Meant to be used with [obsidian.md](https://obsidian.md), or as a neovim plugin [obsidian.nvim](https://github.com/obsidian-nvim/obsidian.nvim)

![DevNote|100](./assests/DevNote-LightYagami.png)

Not the best Arrangement that I came up with, but here a quick overview of what you can expect for each folder

```
DevNote
├── assests             Images that mainly used by the README file
├── Cheatsheet          Cheatsheet and references materials
├── Commands            Like Cheatsheet but for cli tools only, kinda like a tldr page
├── Datasheets          Technical datasheets for electronic components
├── Guides              Comprehensive guides/tutorials for various topics and use cases.
├── Keymaps             Keyboard shortcuts and keymap references
├── Pinouts             Hardware pinout diagrams
├── Random              Random bullsh*t goes here
├── References          Like Guides but more specific to a single tool or concepts.
├── Schematic           Circuit diagrams
└── Specifications      Technical specifications.
```
