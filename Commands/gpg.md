# gpg

The `gpg` command and the commands that come with it, including: `gpgconf`, `gpg-agent`, `gpg-connect-agent`

## Reload config

```sh
gpgconf --reload gpg-agent
gpgconf --list-options gpg-agent
```

## List all gpg components

```sh
gpgconf --list-components
```
